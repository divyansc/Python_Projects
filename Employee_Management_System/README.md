# Employee Management System

This is a simple **Employee Management System** project built in Python. It allows you to manage employee data such as adding, updating, viewing, and deleting employee records. 

## Project Structure

- `company_api/` - Main application code.
- `test/` - Contains test scripts for the application.

## Features

- Add new employees
- Update employee details
- Delete employees
- View employee information

## Requirements

- Python 3.x
- (Optional) Virtual environment for dependency management
